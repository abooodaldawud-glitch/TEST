# Gift Experience

مشروع ويب تفاعلي رومانسي ولطيف، يعمل بدون Backend أو قاعدة بيانات، ومجهز للرفع مباشرة على GitHub Pages.

## طريقة التشغيل

افتح الملف التالي مباشرة في المتصفح:

```text
index.html
```

## هيكل الملفات

```text
index.html
style.css
script.js
assets/images/
assets/videos/
```

## كلمة السر

```text
230806
```

## استبدال الصور

ضع الصور داخل:

```text
assets/images
```

المسارات المستخدمة حاليًا:

```text
assets/images/moon-puzzle.svg
assets/images/her-puzzle.svg
```

يمكنك استبدال الملفات بنفس الأسماء، أو تعديل المسارات من داخل `script.js` و`style.css`.

## ملاحظات

- الموقع مصمم كتجربة Single Page App بسيطة.
- جميع النصوص الظاهرة تمت مراجعتها وتوحيدها حسب النسخة النهائية المرسلة.
- البازل مكوّن من 4 قطع ويعمل بالسحب والإفلات على الكمبيوتر والهاتف.
- عند رفع المشروع على GitHub Pages، ارفع محتويات المجلد كما هي.
